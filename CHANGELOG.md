# Change Log

All notable changes to the "Fix Editor Context Menu" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.2] - 2024-04-19

### Changed
- Renamed extension from "Toggle Comment Context Menu" to "Fix Editor Context Menu"
- Updated package.json metadata for GitHub publication
- Added Cut, Copy, Paste commands to the context menu
- Improved README with detailed feature description
- Updated keywords for better discoverability

## [0.1.1] - Previous Release

### Added
- Initial release with Toggle Line Comment and Toggle Block Comment in context menu
